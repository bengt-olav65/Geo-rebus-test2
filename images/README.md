# Images Directory

This directory should contain the rebus images used as backgrounds.

## Required Files

- `rebus2.jpg` - Background image for question 2 (needs to be added)
- `rebus2.webp` - WebP version (generated by optimize-images script)

## Adding Images

1. Add your source images (JPG format) to this directory
2. Run `npm run optimize-images` to generate optimized WebP versions

## Note

The `rebus2.jpg` file is referenced in the HTML but not included in this repository.
Please add your image file here before deploying.
